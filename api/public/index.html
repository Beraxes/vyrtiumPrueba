API Documentation

Welcome to the API documentation. Here you'll find details on how to use Postman to interact with the API for user registration, login, and managing tasks.

1. User Registration

Use the following details to register a new user:

      POST /users/register
      Content-Type: application/json
      Body:
      {
        "email": "example@email.com",
        "username": "exampleUser",
        "password": "examplePassword"
      }

Expected Response:

      {
        "message": "User successfully created"
      }

2. User Login

Use the following details to log in:

      POST /users/login
      Content-Type: application/json
      Body:
      {
        "username": "exampleUser",
        "password": "examplePassword"
      }

Expected Response:

      {
        "access_token": "your.jwt.token.here"
      }

Note: Use the access_token from the response as a Bearer token for authentication in subsequent requests.

3. Task Management

The following endpoints allow you to create, view, update, and delete tasks. Remember to include the Bearer token in the headers:

  • Create Task: 
              POST /tasks
          Headers: Authorization: Bearer 
          Body:
          {
            "title": "Task Title",
            "description": "Task Description"
          }
  • Get User Tasks: 
              GET /tasks
          Headers: Authorization: Bearer
  • Update Task: 
              PATCH /tasks/:id
          Headers: Authorization: Bearer 
          Body:
          {
            "title": "Updated Title",
            "description": "Updated Description"
          }
  • Delete Task: 
              DELETE /tasks/:id
          Headers: Authorization: Bearer

If you have any questions or issues, feel free to contact the API administrator.
